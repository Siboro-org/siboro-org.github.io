## Reference Documents Backup Convention

- Store each reference work in its own specially named directory under this `referencedocs` directory.
- Use a lowercase, hyphen-separated directory slug in `[author]-[docname]-[year]` order, for example `jtideman-simeloengoen-1922` or `rsinaga-leluhurmargamargabatak-1995`.
- Store the backed-up source file(s) inside the work-specific directory. A work may contain multiple files or formats, such as PDF, DOCX, and extracted TXT.
- Maintain the catalog in `README.md` in this directory.
- Add one Markdown entry per work to `README.md`, including the human-readable title, author, year, a link to the work directory, and links to the stored file(s). Use numbered links when one work has multiple files.
